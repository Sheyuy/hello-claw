title: "Claw University: A Note Before We Begin"
---

Claw University is a night school for people who already have their lobster up and running.

After installing a few Skills, most people run into the same problem:
You can install them, you can run commands, but it never quite feels like "it's actually doing more things for me."
The Skills marketplace is like a giant supermarket — shelf after shelf, easy to browse until your legs give out, yet hard to know which few things to bring home first.

Claw University does one thing: pick a handful of representative scenarios and clearly explain "how to start using these together."
Each article is a concrete thread: from the use case, to which Skills to install, to how to wire them into a small, reusable workflow.

To browse Skills, head directly to [ClawHub](https://clawhub.ai/) or [Chinese ClawHub (Tencent SkillHub)](https://skillhub.tencent.com/#categories);
for the overall map and course-selection thinking, return to the [Claw University homepage (Skills Elective Guide)](/en/university/).

The one-line summary for the whole section:
Install less, seriously work through a few real scenarios, and make your lobster "reliable and useful."
